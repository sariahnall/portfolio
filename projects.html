My Projects

A showcase of the different projects I have worked on in my college time

Website

World Wide Rafting

I created a fully responsive website for a rafting company. I used HTML and CSS to design a clean, easy-to-use layout with features like a navigation menu, customer reviews, and a booking form. The site was built to be visually appealing while also being simple to navigate, so visitors could quickly find the information they needed. This project helped me improve my skills in front-end web development, focusing on making websites that look good and are easy for users to interact with.

Coding

Word guessing game

I built a fun word-guessing game in Python where users try to guess a secret word, getting hints after each guess. The game checks each letter the user guesses and gives feedback on whether it's right or wrong, and even hints at the correct letters. I used loops, conditionals, and basic string handling to create the game logic. Through this project, I learned more about how to structure programs, handle user input, and create interactive experiences using Python. It also helped me practice working with algorithms and simple game design.

Virtual Circuit

Virtual 2-bit Subtractor

I designed a 2-bit subtractor circuit using Tinkercad and tested it with an Arduino. The circuit performs binary subtraction on two 2-bit numbers and shows the result, including the difference and borrow. I used logic gates (AND, OR, XOR, NOT) to build the circuit and programmed the Arduino to input binary numbers and display the output. This project gave me hands-on experience with digital logic design and circuit simulation helping me understand how to apply digital concepts in real-world situations.
